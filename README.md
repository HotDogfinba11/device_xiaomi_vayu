Device configuration for Xiaomi POCO X3 Pro
=========================================

The Xiaomi POCO X3 Pro (codenamed _"vayu"_ , _"bhima"_) is a high-end smartphone from Xiaomi.

It was announced in March 2021 22. Release date was in March 2021 24.

## Device specifications

Basic   | Spec Sheet
-------:|:-------------------------
CPU     | Octa-core (1x2.96 GHz Kryo 485 Gold & 3x2.42 GHz Kryo 485 Gold & 4x1.78 GHz Kryo 485 Silver)
Chipset | Qualcomm Snapdragon 860
GPU     | Adreno 640
Memory  | 6/8 GB RAM
Shipped Android Version | 11.0
Storage | 128/256 GB (UFS 3.1)
Battery | Li-Po 5160 mAh, non-removable
Display | 1080 x 2400 pixels, 6.67 inches (~395 ppi pixel density)
Camera  | 48MP(Primary), 8MP(Ultrawide), 2MP(Macro), 2MP(Depth), dual-LED(dual tone) flash, 20MP(Selfie)
